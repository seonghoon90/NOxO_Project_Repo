## 💡 핵심 인사이트

### 1. 간단함이 승리함

Ridge Regression이 모든 복잡한 모델을 능가했습니다.

**왜?**
- NOx 예측 문제가 본질적으로 선형 구조
- 충분한 피처 수로 선형 모델도 좋은 표현력
- 데이터가 깨끗해서 정규화만으로 충분

### 2. 트리 모델의 과적합

GradientBoosting, XGBoost는 Train 성능은 높지만 Test에서 떨어집니다.

```
XGBoost Train MAE:  0.0976
XGBoost Test MAE:   0.1506
갭: 0.053 (54% 증가!)

Ridge Train MAE:    0.1517
Ridge Test MAE:     0.1479
갭: -0.0038 (오히려 개선!)
```

→ **Ridge가 더 견고함**

### 3. 속도의 중요성

Ridge는 거의 즉시 학습됩니다.

```
실시간 예측 필요 시:
  Ridge: 0.004초 ← 배포 최적
  LightGBM: 0.5초
  GradientBoosting: 48초 (부적절)
```
